# 🎓 Everest College Event Management System (ECEMS)

A complete event management system for Everest College with registration, filtering, and admin features.

## Features
✅ User Registration & Login
✅ Event Discovery
✅ Event Registration
✅ User Dashboard
✅ Admin Panel
✅ Email Notifications
✅ Responsive Design

## Setup
1. Extract to C:\xampp\htdocs\ecems\
2. Create database 'ecems' in phpMyAdmin
3. Import database/ecems-database.sql
4. Start XAMPP
5. Visit http://localhost/ecems/
